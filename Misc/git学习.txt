git status:状态
git diff readme.txt:查看差别
git add readme.txt:添加至暂存区
git commit -m "update content":把暂存区文件提交至master
git log:日志
git log --pretty=oneline:精简日志
git reset --hard HEAD^:返回上一版本
cat readme.txt:查看文件
git reset --hard 1094a:移动到某一版本
git reflog:操作日志
git rm test.txt:删除文件，可以用add代替（之前要先手动删除）
git checkout -- test.txt:用版本库里的版本替换工作区的版本
