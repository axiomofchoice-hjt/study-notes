## 1. 配置

`~/.vimrc`

```text
set number
set syntax=on
set tabstop=4
set softtabstop=4
set shiftwidth=4
set ruler
set autoindent
set cindent
set expandtab
set showcmd
set scrolloff=3
set foldmethod=manual
set nocompatible
set completeopt=preview,menu
set nobackup
set magic
set noeb
set backspace=2
set mouse=a
set selection=exclusive
set selectmode=mouse,key
set showmatch
set smartindent
```

## 2. 常用操作

- 清空 `ggdG`
- 保存并退出 `:wq` / `ZZ`
- 转到第 100 行 `:100`
