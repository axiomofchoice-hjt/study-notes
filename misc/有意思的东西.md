# 有意思的东西

CS

- qemu：全虚拟化平台
- cntlm：二次代理
- valgrind：内存检测工具
- broot：linux 命令行文件游览器
- just：类似 makefile 的构建工具

其他

- AutoHotkey：全局热键
- Snipaste：截屏
- PowerToys：win 工具箱
- Manim：3b1b 的数学动画引擎
- foam：vscode 笔记插件
- excalidraw：手绘风格绘图工具
- WakaTime：记录工作时间
- Latex Sympy Calculator：计算 Latex 表达式
- graph editor：画图（图论）
